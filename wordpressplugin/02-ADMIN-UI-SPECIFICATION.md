# FlowChat Pro - Admin UI Specification

**Version:** 1.0.0  
**Last Updated:** December 2024  
**Status:** Technical Specification

---

## Design Philosophy

### Guiding Principles

1. **Non-technical friendly**: No code, no confusion
2. **Progressive disclosure**: Simple defaults, advanced options hidden
3. **Visual feedback**: Live previews, instant validation
4. **Consistent patterns**: Same interactions everywhere
5. **WordPress native feel**: Uses WordPress UI patterns and colors

### Target User Persona

**Maria, Marketing Manager**
- Comfortable with WordPress admin
- Not a developer
- Wants to set up a chatbot in < 5 minutes
- Needs to customize colors to match brand

## Mobile Responsive Admin

### Tablet View (768px - 1024px)

- Tabs stack horizontally below content
- Live preview moves below form
- Two-column grids become single column

### Mobile View (< 768px)

- Full single-column layout
- Collapsible sections
- Bottom-fixed save button
- Preview becomes expandable modal

## Accessibility Requirements

### WCAG 2.1 AA Compliance

1. **Color contrast**: All text 4.5:1 minimum
2. **Focus indicators**: Visible on all interactive elements
3. **Keyboard navigation**: Full admin accessible via keyboard
4. **Screen readers**: All form fields have proper labels
5. **Error states**: Clear error messages with aria-describedby
